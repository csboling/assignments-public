\section*{Problem \#4.7}
A capacitor's contribution to the delay
is determined by the resistances that are shared by the path
to that capacitor and the path to the load, so we have
\begin{align*}
\tau_{DB}
&=
    R_1                     C_1
  + R_1                     C_2
  + (R_1 + R_3)             C_3
  + R_1                     C_4
  + (R_1 + R_3)             C_5
  + (R_1 + R_3 + R_6)        C_6
  + (R_1 + R_3)             C_7
  + (R_1 + R_3 + R_6 + R_8) C_8 \\
&= 2.525 \times 10^{-10} ~\mathrm{s}
 = 0.25 ~\mathrm{ns}.
\end{align*}
